    Project Overview

    This submission implements a multi‑page café website (Riwaq Café) with an external CSS stylesheet and embedded images. The site is organized into a Home page, full Menu page, Order form, and Contact page.

    Requirements Checklist

    • ✔ Use multiple HTML files → index.html, menu.html, order.html, contact.html.
    • ✔ Add images → Images displayed on Home (and available for Menu), stored in images/.
    • ✔ Use external style sheet → All pages link to style.css.

    Site Map (Links)

    • Home – Landing with featured images.
    • Menu – Coffee, Tea, Cold Drinks, Pastries tables.
    • Order – Customer form (name, phone, email, category, item, size, add‑ons, quantity, notes).
    • Contact – Opening hours, phone, email.

    Testing Notes

    • Navigation links allow moving between all pages without errors.
    • Images paths: images/latte.jpg, images/cappuccino.jpg, images/croissant.jpg, images/lemonade.jpg, logo images/Riwaq.jpg.
    • Form fields in Order include required Name and sensible defaults (e.g., Medium size, quantity ≥ 1).
    • CSS color typo fixed: #cfc6b8 (replaces the mistaken #cfc6ب8).

    How to Run

    1. Open project1.html to see this overview and use the links, or open index.html directly.
    2. Ensure the images/ folder contains the listed images with the exact names.
